BMI-Tracker is a simple app to help you monitor your body mass index (BMI). It saves your data on your device, shows charts of your progress, and lets you export your records as CSV files. You don’t need any technical skill to use it.

## 📋 What is BMI-Tracker?

BMI-Tracker is a small application built with React and Vite. It runs entirely in your web browser on Windows. It stores your weight and height data locally on your computer. This means your information stays private unless you share it. You can also view your changes over time using clear charts. If you want, you can export your data to a CSV file to review it in spreadsheet programs like Excel.

The app works offline once you open it, so you don’t need to be connected to the internet after the first load.

## ❓ Troubleshooting

### The app doesn’t open or load

- Make sure you use a supported browser: Chrome, Edge, or Firefox.

- If running offline from a downloaded folder, check you opened the `index.html` file correctly.

### Calculations look wrong

- Double-check that you entered weight and height using the right units.

- Ensure all fields are filled before pressing **Calculate**.

### Charts do not update

- Reload the page or restart the browser.

- Check that local storage on your browser is not disabled.
